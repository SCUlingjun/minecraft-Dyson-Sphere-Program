# energy

能源系统根目录。

本 mod 使用自定义能源单位 **MJ（Mega Joule）**，兼容 NeoForge 的 `IEnergyStorage` 接口以便与其他 mod 互通。

核心概念：
- **生产者**：发电机、太阳能板、戴森球接收站 → 每刻向网络注入 MJ
- **消费者**：机器、物流机器人充电站、武器 → 每刻从网络抽取 MJ
- **存储**：蓄电池、超级电容 → 缓冲网络盈余/缺口

顶层类：
- `EnergyUnit.java` — 能源单位常量与换算工具
- `EnergyHelper.java` — 静态工具方法（向相邻方块分发能量等）

子目录：
- `network/` — 能源网络的拓扑管理与流量调度
- `storage/` — 各类能源存储容器的实现
