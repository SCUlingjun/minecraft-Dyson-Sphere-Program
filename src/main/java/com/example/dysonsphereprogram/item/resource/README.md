# item/resource

原材料与资源物品目录。

包含从世界中采集或机器加工得到的基础资源，是整个科技树的原料来源。

分类：
- 矿石产物：`IronIngotDSPItem`、`CopperIngotItem`、`SiliconItem`、`TitaniumItem`
- 气体资源（瓶装）：`HydrogenBottleItem`、`DeuteriumBottleItem`、`AntimatterItem`
- 有机资源：`PlantFiberItem`、`OrganicCrystalItem`
- 特殊资源：`StrangeMatterItem`（黑洞物质，戴森球终极原料）

气体类物品通常也有对应的流体形态（见 `fluid/`），瓶装版本仅用于少量存储和玩家背包携带。
