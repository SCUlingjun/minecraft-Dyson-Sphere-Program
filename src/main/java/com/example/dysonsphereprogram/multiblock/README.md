# multiblock

多方块结构（Multiblock）框架根目录。

本 mod 的大型设备（戴森球发射井、矩阵研究站、行星发动机等）由多个方块按固定形状拼装而成，统一由此框架驱动。

核心流程：
1. 玩家放置控制器方块后，`controller/` 中的控制器 BlockEntity 触发结构检测
2. `validation/` 中的检测器按 `pattern/` 定义的模板扫描周围方块
3. 检测通过后，控制器进入运行状态，驱动整体逻辑（消耗能量、推进进度）
4. 任意组成方块被破坏时，控制器收到通知并中断运行

子目录：
- `pattern/` — 结构模板定义（形状、层数、方块类型映射）
- `controller/` — 控制器基类与各大型设备的控制器实现
- `validation/` — 结构完整性检测与错误位置反馈
