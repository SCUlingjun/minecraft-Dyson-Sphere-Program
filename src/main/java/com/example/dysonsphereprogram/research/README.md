# research

科技树系统根目录。

本 mod 的科技树参考戴森球计划原作，分为六大矩阵（信息、结构、能源、重力、电磁、宇宙）。玩家在矩阵研究站中消耗对应矩阵合成对应科技，逐步解锁更高级的配方和机器。

顶层类：
- `ResearchManager.java` — 全局科技进度管理器，以 `SavedData` 持久化每位玩家的解锁状态
- `ResearchData.java` — 单个玩家的科技解锁数据对象
- `MatrixType.java` — 枚举六种矩阵类型及对应物品引用

子目录：
- `tree/` — 科技树节点定义（每项科技的前置、消耗、解锁内容）
- `unlock/` — 解锁事件处理（解锁新配方、解锁新方块放置权限等）
