# assets/dysonsphereprogram/textures/item

物品贴图目录。

存放所有物品的 `.png` 图标，标准尺寸 16×16（Minecraft UI 基准网格单元）。

分类命名建议：
- 原材料：`silicon.png`、`titanium_ingot.png`
- 组件：`circuit_board_mk1.png`、`circuit_board_mk2.png`
- 工具：`electromagnetic_rifle.png`、`jetpack.png`
- 矩阵：`electromagnetic_matrix.png`、`energy_matrix.png`

同一物品的多个状态（如电量不同的电池）命名为 `battery_0.png`、`battery_1.png`，通过物品模型 `overrides` 切换。
